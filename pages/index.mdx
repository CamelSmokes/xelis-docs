# Welcome to Xelis docs

This comprehensive guide is designed to help developers and users
understand the Xelis platform.

Inside, you'll find detailed information on the core features and functionality of the Xelis Blockchain,
including its consensus algorithm, smart contract capabilities, and privacy measures.

You'll also learn how to set up and configure your own Xelis node, interact with the blockchain APIs and other tools.
Maybe develop your own decentralized applications :)

Whether you're an experienced blockchain developer or just getting started,
this documentation is the official resource for working with the Xelis Blockchain.

Dive in and discover the permissionless decentralized technology of Xelis!
