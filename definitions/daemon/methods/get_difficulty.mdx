### `get_difficulty`

> Retrieve current difficulty and associated network hashrate.

Request

```json
{
  "jsonrpc": "2.0",
  "method": "get_difficulty",
  "id": 1
}
```

Response

```json
{
  "id": 1,
  "jsonrpc": "2.0",
  "result": {
    "difficulty": "79746345000",
    "hashrate": "5316423000",
    "hashrate_formatted": "5.32 GH/s"
  }
}
```
